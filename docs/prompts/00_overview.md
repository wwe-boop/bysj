# 提示词总览（精简版）

本项目围绕“融合定位 + 智能调度”的一体化系统展开：DRL实时准入控制 × DSROQ资源分配 × 融合定位（CRLB/GDOP）× Hypatia仿真，目标是实现“QoE × 定位质量”的联合最优。

## 产出物
- 论文文档（docs/）
- 系统设计文档（design/）
- 可运行原型（src/）

- 部署方案（deploy/）
