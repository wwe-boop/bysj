# 第9章 结论与展望

## 9.1 研究工作总结

提出面向LEO卫星网络的DRL准入控制，与DSROQ形成分层协同，实现QoE驱动的全局优化并在Hypatia上完成高保真验证。

## 9.2 主要贡献

- 首个面向LEO的实时DRL准入控制与时间感知状态空间。
- 准入-分配分层架构与细粒度动作空间。
- 多维度实验、消融与敏感性评估，展现综合优势。

## 9.3 不足与局限

- 泛化与跨星座迁移能力仍需强化；
- 训练成本与收敛速度受限于仿真复杂度；
- 现实系统中的在线约束与安全性尚需进一步验证。

## 9.4 未来工作方向

- 联邦/分布式强化学习；
- 在线/自适应与安全约束的策略优化；
- 多域协同（空天地海）与跨域QoE管理；
- 真实轨道与测量数据驱动的校准与部署。
